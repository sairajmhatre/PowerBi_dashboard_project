# Power Bi_dashboard_project
This is power bi end to end project !!   
Used complex parameters to drill down in worksheet and customization using filters and Slicers Created connections, join new tables, calculations to manipulate data and enable user driven parameters for visualizations Used different types of customized visualization (bar chart, pie chart, donut chart, clustered  bar chart, scatter chart, line chart, area chart, map, slicers, etc)
